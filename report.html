05 — Research Workflow & Refinement Loop

How the dataset was built

An agent-driven first pass was checked through a separate ChatGPT review loop with explicit citations and structured outputs.

Agent pipeline

  • Research: 50 app records processed using Composio search context and GPT-4o structured output.
  • Extraction: Auth, access tier, API surface, verdict, blocker, and evidence URL captured in a fixed schema.
  • Controls: Checkpoints and validation rules prevent failed requests from becoming false negative findings.

Verification design

  • Evidence review: ChatGPT compared every completed record against cited official documentation.
  • Structured import: Results were required to provide field-level pass/fail/partial status and source URLs.
  • Refinement: 23 records were updated only where the external review proposed an explicit correction.

Feedback loop

Pass 1 · Agent
Raw research on 50 apps using structured parsing.
Pass 2 · Review
ChatGPT-assisted review against official documentation with citations.
Pass 3 · Refine
23 record corrections applied from the review output.
